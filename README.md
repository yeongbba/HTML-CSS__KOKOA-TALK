# KOKOA-TALK PROJECT
  
  
채팅 서비스 카카오톡의 디자인을 구현을 한 프로젝트입니다.  
카카오톡의 메인 화면, 친구 목록, 세팅, 채팅방 등을 제작하였습니다.
<br>
<br>
  
### STACKS : HTML5, CSS
  
HTML5와 CSS를 주기술로 사용하였습니다.
<br>
<br>  
  
### ACHIEVEMENT GOAL
  
- BEM 방법론으로 CSS Architecture 구성하기  
- FLEX를 활용하여 페이지의 레이아웃 잡기  
- ANIMATION 기능으로 페이지의 역동성을 더하기  
- 미디어쿼리를 이용하여 반응형 웹 만들기
<br>
